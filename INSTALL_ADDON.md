# Встановлення GSM MQTT Gateway як аддону Home Assistant

---

## Варіант А — Локальний аддон (без GitHub)

Найпростіший спосіб: копіюємо файли прямо в HA через File Editor.

### Крок 1 — Створити папку аддону

1. Відкрий **File editor** у HA
2. Перейди в корінь `/`
3. Створи папку: `addons`
4. Всередині `addons` створи папку: `gsm_gateway`

### Крок 2 — Завантажити файли

Перейди в папку `/addons/gsm_gateway/` і створи 4 файли:

| Файл | Що це |
|------|-------|
| `config.yaml` | Маніфест аддону |
| `Dockerfile` | Опис Docker контейнера |
| `run.sh` | Скрипт запуску |
| `gateway.py` | Основний код шлюзу |
| `webui.py` | Веб-інтерфейс |

Скопіюй вміст кожного файлу з папки що ти завантажив.

### Крок 3 — Встановити аддон

1. Перейди: **Налаштування → Додатки (Settings → Add-ons)**
2. Натисни **три крапки (...)** у правому верхньому куті
3. Вибери **«Reload»** — HA знайде локальний аддон
4. У списку аддонів знайди **«GSM MQTT Gateway»**
5. Натисни **ВСТАНОВИТИ**

## Варіант Б — GitHub репозиторій (рекомендовано)

Дозволяє оновлювати аддон одним кліком.

### Крок 1 — Створити GitHub репозиторій

1. Зайди на [github.com](https://github.com) і створи новий репозиторій
2. Назви його, наприклад: `ha-gsm-gateway`
3. Зроби його **Public**
4. Завантаж всі файли аддону у такій структурі:

```
ha-gsm-gateway/              ← корінь репозиторію
├── repository.yaml
└── gsm_gateway/
    ├── config.yaml
    ├── Dockerfile
    ├── run.sh
    ├── gateway.py
    └── webui.py
```

> Для завантаження файлів: на GitHub натисни **Add file → Upload files**

### Крок 2 — Додати репозиторій в HA

1. Перейди: **Налаштування → Додатки → Магазин додатків**
2. Натисни **три крапки (...)** у правому верхньому куті
3. Вибери **«Repositories»**
4. В поле введи URL свого репозиторію:
   ```
   https://github.com/YOUR-USERNAME/ha-gsm-gateway
   ```
5. Натисни **ADD**
6. Закрий вікно і оновіть сторінку

### Крок 3 — Встановити аддон

1. У магазині додатків прокрути вниз — знайди секцію **«GSM MQTT Gateway»**
2. Клікни на аддон → **ВСТАНОВИТИ**
3. Зачекай поки збереться Docker образ (2-5 хвилин)

## Налаштування аддону

Після встановлення перейди на вкладку **Конфігурація (Configuration)**:

| Параметр | Опис | Приклад |
|----------|------|---------|
| `serial_port` | Порт USB модему | `/dev/ttyUSB0` |
| `serial_baudrate` | Швидкість порту | `115200` |
| `mqtt_host` | IP MQTT брокера | `192.168.1.100` або `core-mosquitto` |
| `mqtt_port` | Порт MQTT | `1883` |
| `mqtt_username` | Логін MQTT | `user` |
| `mqtt_password` | Пароль MQTT | `password` |
| `trusted_numbers` | Довірені номери | `["+380501234567"]` |
| `status_interval` | Інтервал статусу (сек) | `60` |
| `log_level` | Рівень логування | `INFO` |

> Якщо використовуєш вбудований Mosquitto аддон — в `mqtt_host` вкажи `core-mosquitto`

Після налаштування натисни **ЗБЕРЕГТИ → ЗАПУСТИТИ**.

## Веб-інтерфейс

Після запуску аддону відкрий вкладку **«Веб-інтерфейс» (Web UI)** —
або перейди за адресою: `http://YOUR-HA-IP:8099`

Там побачиш:
- 🟢 Статус модему (онлайн/офлайн)
- 📶 Рівень сигналу (dBm)
- 📡 Оператор і тип реєстрації
- 💳 Стан SIM карти
- 📋 Живий журнал подій (оновлення кожні 10 сек)

---

## Визначення порту USB модему

Якщо не знаєш порт модему:
1. **Налаштування → Система → Обладнання → Всі пристрої**
2. Знайди SIM800 — зазвичай `/dev/ttyUSB0` або `/dev/ttyACM0`

---

## Перевірка роботи

1. Перейди: **Інструменти розробника → MQTT**
2. Підпишись на топік `gsm/#`
3. Надішли SMS на SIM картку
4. У MQTT має з'явитись:

```json
{
  "from": "+380501234567",
  "text": "Тест",
  "trusted": 1,
  "timestamp": "2024-01-01T12:00:00Z"
}
```
